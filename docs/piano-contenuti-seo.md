### 1.2. Выводы по ландшафту

1. **Ниша «скорость по контракту» свободна.** Скорость как USP продают (SitoWP),
   но письменную гарантию с неустойкой — никто из найденных. Это подтверждает
   ядро позиционирования.
2. **Прозрачные цены — гигиенический фактор топа.** Все, кто ранжируется по
   «quanto costa…», публикуют конкретные вилки. Наши цены (€1.900–2.800 /
   €3.900–5.800 / €7.500–14.000) попадают в рыночный коридор **[✓]**
   (Tready: PMI-сайт €3–8K; DSI: €2,5–8K) — мы не дороже рынка, при этом
   даём больше гарантий. Это надо говорить прямо.
3. **По запросам «migliori web agency…» ранжируются каталоги** (Sortlist),
   а не агентства — кластер «migliori» в органике не атакуем, вместо этого
   присутствуем в самих каталогах (Sortlist, Clutch) как канал.
4. **PWA-выдача в Италии полупустая:** по «progressive web app Italia» в топе
   каталог Sortlist, а не агентства. Наша страница /servizi/siti-pwa/ может
   занять нишу малой кровью — усилить и обвязать статьями.
5. **Подписка как продукт** (Italia Multimedia, €190/мес) — рынок приучен
   к ежемесячной оплате «сопровождения». Наш retainer «Crescita continua»
   (€390–900/мес из концепции) продаваем, но надо показать больше ценности
   за большую цену (KPI-отчёты, CRO, контент).

## 2. Что реально ранжируется в этой нише (форматы)

Проверенная связка форматов итальянского SERP этой ниши **[✓]** (архитектура
DSI Design воспроизводится у Tready, Webcom, Noonic):

1. **Городской лендинг** «Realizzazione siti web {città}» — коммерческий интент.
2. **Ежегодный прайс-гайд** «Quanto costa un sito web nel {anno}» — стабильный
   URL, обновляемый заголовок и контент раз в год **[✓]** (у DSI slug 2025,
   title 2026).
3. **Отраслевые страницы** (siti per centri estetici / studi legali / PMI…).
4. **Бесплатные инструменты** — SEO Tester Online стабильно в топе по «analisi
   SEO gratis»: формат работает, но по этому конкретному ключу конкуренция
   высокая — инструменты используем как лид-магниты и линкабельные активы,
   а не как самостоятельную SEO-ставку.
5. **Google Business Profile** — для локального интента (46% поисков имеют
   локальный интент **[~]**). GBP-результаты приходят за 3–6 мес против
   12–18 мес по национальным ключам **[✓]** — самый быстрый канал для нового
   сайта.

Ограничение на городские лендинги: **15–20 содержательных страниц максимум,
не сотни шаблонных** — против doorway-политики Google **[~]** (Search Engine
Land). Каждая страница — с локальными кейсами, отзывами с гео-привязкой, NAP.

Длиннохвостые запросы ≈ 91,8% всех запросов Google **[~]** (SEOZoom) — при
нулевом возрасте домена основной трафик первых месяцев придёт именно с них.

Сезонность: подтверждённых публикаций с данными по нише нет; общий паттерн
итальянских B2B-услуг — пики сентябрь–январь, провал в августе (ferie).
Действие: проверить в Google Trends (geo=IT) «realizzazione siti web» перед
планированием публикаций; в августе — не запускать кампании, но готовить
сентябрьский контент. В GBP обязательно ставить orari speciali на август **[~]**.

### 4.4. Google Business Profile (параллельно, недели 1–4)

Самый быстрый канал для нового сайта (**[✓]** локальная видимость за 3–6 мес
против 12–18 национальной):

1. Завести/заполнить профиль на 100%: категория Web designer/Agenzia web,
   Via Andrea Solari 43, часы, услуги, фото офиса, ссылка на LinkedIn (B2B).
2. Пост в профиль раз в неделю (репост статей блога).
3. Просить отзыв у каждого клиента после сдачи (у DSI ~100 отзывов — это
   их главный актив; нам нужно системно догонять).
4. Orari speciali на август/праздники — обязательный пункт чек-листа.
5. NAP (название-адрес-телефон) одинаковый на сайте, GBP и в каталогах.

1. **Test velocità → реальный PSI API** (сейчас демо) — приоритет 1: это

4. **Analisi SEO** — приоритет 4: SERP занят SEO Tester Online; делать

- Внутренняя перелинковка pillar–cluster (каждая статья → услуга и инструмент).

## 5. Чего мы НЕ делаем (и почему)

1. Не гонимся за «migliori web agency milano» — там каталоги.
2. Не клонируем 50 городских страниц — doorway-риск; максимум 6–8 в год 1.
3. Не публикуем статьи без глубины ради частоты — E-E-A-T дороже объёма.
4. Не строим SEO-ставку на «analisi SEO gratis» — SERP занят инструментом
   с многолетним возрастом.
5. Не запускаем платный трафик до реального PSI API в test velocità: обещание
   «мы измеряем» с демо-инструментом — репутационный риск.
