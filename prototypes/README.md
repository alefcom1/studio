# prototypes/ — мёртвые прототипы (НЕ продакшн)

Здесь лежит ранний **дизайн-прототип на Astro** (`astro/`), с которого начинался
проект. Он **не является продакшн-сайтом** и никуда не деплоится.

Боевой продукт — это **дочерняя тема WordPress** в `wordpress/remarka-studio/`
(+ генератор страниц в `wordpress/build-tools/`). Живой сайт remarka.biz работает
на ней (Hetzner). Все правки контента и кода вносим только туда.

## Зачем оставили, а не удалили

Прототип держим как дизайн-референс (макеты блоков, токены, анимации), с которого
портировали тему. История в git тоже сохранена.

## ⚠️ Важно: в прототипе устаревшие и выдуманные данные

Astro-версия содержит **placeholder-реквизиты и вымышленные факты**, которые давно
вычищены из боевого сайта: тестовый адрес `Via Andrea Solari 43`, выдуманная
`P.IVA 12345678901`, фейковые «Recensioni Google», нерабочая форма
(«заявка отправлена» без отправки), переключатель IT/EN/RU без перевода контента.

Поэтому:

- **НЕ собирать и НЕ публиковать** этот прототип (`npm run build` внутри `astro/`
  соберёт сайт с фейковыми данными).
- Использовать только для просмотра дизайна локально при необходимости.

Если прототип больше не нужен как референс — папку `prototypes/` можно удалить
целиком, боевой сайт от этого не зависит.
